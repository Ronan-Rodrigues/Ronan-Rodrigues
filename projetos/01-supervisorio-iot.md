\# 🏭 Supervisorio IoT Industrial



\## Link Principal

🔗 \[github.com/Ronan-Rodrigues/supervisorio-iot-industrial](https://github.com/Ronan-Rodrigues/supervisorio-iot-industrial)



\## Conceito

Substituição de supervisórios desktop (Elipse E3) por dashboard web moderno, 

com dados reais de sensores ESP32 via MQTT.



\## Stack Completo

| Camada | Tecnologia |

|--------|-----------|

| Hardware | ESP32 + DHT22 + BMP280 |

| Firmware | C++ (Arduino) |

| Comunicação | MQTT (HiveMQ) |

| Backend | Node.js + Express + SQLite |

| Frontend | HTML5 + Chart.js |

| Automação | n8n (alertas) |

| DevOps | Docker + VPS Cloud |



\## Funcionalidades Principais

\- \[x] Dashboard web em tempo real

\- \[x] Coleta de dados de sensores via ESP32

\- \[x] Broker MQTT local ou cloud

\- \[x] Banco de dados SQLite para histórico

\- \[x] Alertas por email/WhatsApp quando valores saem da faixa

\- \[x] Exportação de relatórios CSV/Excel

\- \[x] Deploy containerizado em Docker



\## Aprendizados

\- MQTT mais leve que Modbus/Profinet para IoT

\- Dashboard web acessível de qualquer dispositivo

\- SQLite robusto para aplicações pequenas/médias
